# yt-ex-c-pb
Script buat ambil data video trending untuk setiap negara yang berubah setiap hari.


## Setup
1. Isi API key di main.py
   api_key = "YOUR_KEY"
2. ganti path
```
.github/workflows-template
``` 
menjadi 
```
.github/workflows
```

## Git Clone
```
git clone https://github.com/Mayasadev/yt-ex-c-pb.git
mv yt-ex-c-pb/.github/workflows-template yt-ex-c-pb/.github/workflows

```
